ECommerce Platform

Платформа для обработки заказов на микросервисной архитектуре. Учебный проект.

## Технологии

- **Java 21**, Spring Boot 3.3, Spring Cloud Gateway
- **PostgreSQL 15** — пользователи, заказы, запасы
- **MongoDB 7** — каталог товаров, уведомления
- **Redis 7** — кеширование
- **Kafka** + Schema Registry (Avro) — события между сервисами
- **gRPC** — быстрое общение order-service ↔ inventory-service
- **Prometheus + Grafana** — метрики и дашборды
- **Docker, Docker Compose** — контейнеризация
- **GitHub Actions** — автотесты при пуше

## Как запустить

### 1. Поднять инфраструктуру
```powershell
docker-compose up -d postgres mongodb kafka zookeeper schema-registry prometheus grafana
